Node package manager

Node package manager or npm is a free package manager and installer for Node.js. With npm, you can donwnload and manage dependencies of your project. Also, it is world's largest software library, which contains over 800 000 packages. There is a special website called npmjs.com where you can search for the packages that you need for your project. But since everyone can publish their own packages, make sure to check that the package is safe to use. The easiest way to check it is to look at the weekly downloads count - the more downloads the better.

How to initialize npm package?

If you have a project, and you want to quickly set up your npm package, you can use npm init in your command prompt. But before that make sure that you are in the project folder. Use cd command to easily navigate in the command prompt. When initializing through npm init, you can provide various information about the project, like: package name, version, description, entry point, test command, git repository, keywords, author and license. Then, npm will create package.json and other files needed for dependency managment.

What is package.json?

A package.json file:
  • lists the packages your project depends on
  • specifies versions of a package that your project can use using semantic versioning rules
  • makes your build reproducible, and therefore easier to share with other developers

Dependencies that we used and how to install them

In our classes we have used nodemon and express, but since I had some problems with nodemon that I could not fix, I have used supervisor instead.

What are these packages for?
  • nodemon helps develop node.js based applications by automatically restarting the node application when file changes in the directory are detected.
  • express is a back end web application framework for Node.js, which is designed for building web applications and APIs. You can find more about express here
  • supervisor is a nodemon alternative, it restarts programs when a file changes.
To install the package, type npm i package-name, and don't forget to be in your project's directory. To uninstall the package, type npm un package-name. Also, you can use flag -g to install package globally.

How to use these packages?
  • nodemon type nodemon app-name.js in the CLI. Nodemon will start the program and restart it when changes are detected. You can also specify which files to "listen" for.
